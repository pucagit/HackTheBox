# Introduction to Active Directory Enumeration & Attacks
## Active Directory Explained
`Active Directory` (`AD`) is a directory service for Windows enterprise environments. AD is based on the protocols x.500 and LDAP that came before it and still utilizes these protocols in some form today. It is a distributed, hierarchical structure that allows for centralized management of an organization’s resources, including users, computers, groups, network devices and file shares, group policies, devices, and trusts. AD provides `authentication, accounting, and authorization` functions within a Windows enterprise environment.
