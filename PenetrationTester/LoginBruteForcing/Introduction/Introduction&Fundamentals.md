# Introduction & Fundamentals
## Types of brute force

<table class="bg-neutral-800 text-primary w-full"><thead class="text-left rounded-t-lg"><tr class="border-t-neutral-600 first:border-t-0 border-t"><th class="bg-neutral-700 first:rounded-tl-lg last:rounded-tr-lg p-4">Method</th><th class="bg-neutral-700 first:rounded-tl-lg last:rounded-tr-lg p-4">Description</th><th class="bg-neutral-700 first:rounded-tl-lg last:rounded-tr-lg p-4">Example</th><th class="bg-neutral-700 first:rounded-tl-lg last:rounded-tr-lg p-4">Best Used When...</th></tr></thead><tbody class="font-mono text-sm"><tr class="border-t-neutral-600 first:border-t-0 border-t"><td class="p-4"><code class="bg-neutral-700 mb-6 text-blue-250 py-1 px-1.5">Simple Brute Force</code></td><td class="p-4">Systematically tries all possible combinations of characters within a defined character set and length range.</td><td class="p-4">Trying all combinations of lowercase letters from 'a' to 'z' for passwords of length 4 to 6.</td><td class="p-4">No prior information about the password is available, and computational resources are abundant.</td></tr><tr class="border-t-neutral-600 first:border-t-0 border-t"><td class="p-4"><code class="bg-neutral-700 mb-6 text-blue-250 py-1 px-1.5">Dictionary Attack</code></td><td class="p-4">Uses a pre-compiled list of common words, phrases, and passwords.</td><td class="p-4">Trying passwords from a list like 'rockyou.txt' against a login form.</td><td class="p-4">The target will likely use a weak or easily guessable password based on common patterns.</td></tr><tr class="border-t-neutral-600 first:border-t-0 border-t"><td class="p-4"><code class="bg-neutral-700 mb-6 text-blue-250 py-1 px-1.5">Hybrid Attack</code></td><td class="p-4">Combines elements of simple brute force and dictionary attacks, often appending or prepending characters to dictionary words.</td><td class="p-4">Adding numbers or special characters to the end of words from a dictionary list.</td><td class="p-4">The target might use a slightly modified version of a common password.</td></tr><tr class="border-t-neutral-600 first:border-t-0 border-t"><td class="p-4"><code class="bg-neutral-700 mb-6 text-blue-250 py-1 px-1.5">Credential Stuffing</code></td><td class="p-4">Leverages leaked credentials from one service to attempt access to other services, assuming users reuse passwords.</td><td class="p-4">Using a list of usernames and passwords leaked from a data breach to try logging into various online accounts.</td><td class="p-4">A large set of leaked credentials is available, and the target is suspected of reusing passwords across multiple services.</td></tr><tr class="border-t-neutral-600 first:border-t-0 border-t"><td class="p-4"><code class="bg-neutral-700 mb-6 text-blue-250 py-1 px-1.5">Password Spraying</code></td><td class="p-4">Attempts a small set of commonly used passwords against a large number of usernames.</td><td class="p-4">Trying passwords like 'password123' or 'qwerty' against all usernames in an organization.</td><td class="p-4">Account lockout policies are in place, and the attacker aims to avoid detection by spreading attempts across multiple accounts.</td></tr><tr class="border-t-neutral-600 first:border-t-0 border-t"><td class="p-4"><code class="bg-neutral-700 mb-6 text-blue-250 py-1 px-1.5">Rainbow Table Attack</code></td><td class="p-4">Uses pre-computed tables of password hashes to reverse hashes and recover plaintext passwords quickly.</td><td class="p-4">Pre-computing hashes for all possible passwords of a certain length and character set, then comparing captured hashes against the table to find matches.</td><td class="p-4">A large number of password hashes need to be cracked, and storage space for the rainbow tables is available.</td></tr><tr class="border-t-neutral-600 first:border-t-0 border-t"><td class="p-4"><code class="bg-neutral-700 mb-6 text-blue-250 py-1 px-1.5">Reverse Brute Force</code></td><td class="p-4">Targets a single password against multiple usernames, often used in conjunction with credential stuffing attacks.</td><td class="p-4">Using a leaked password from one service to try logging into multiple accounts with different usernames.</td><td class="p-4">A strong suspicion exists that a particular password is being reused across multiple accounts.</td></tr><tr class="border-t-neutral-600 first:border-t-0 border-t"><td class="p-4"><code class="bg-neutral-700 mb-6 text-blue-250 py-1 px-1.5">Distributed Brute Force</code></td><td class="p-4">Distributes the brute forcing workload across multiple computers or devices to accelerate the process.</td><td class="p-4">Using a cluster of computers to perform a brute-force attack significantly increases the number of combinations that can be tried per second.</td><td class="p-4">The target password or key is highly complex, and a single machine lacks the computational power to crack it within a reasonable timeframe.</td></tr></tbody></table>
